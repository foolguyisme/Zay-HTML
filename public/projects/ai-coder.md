# AI Coder Project

> 本地端 AI 程式助手平台

## 概述

一套完整的本地端程式碼生成與驗證平台，整合 Web UI、Docker 沙盒執行環境與模型微調流程。

## 核心功能

- **程式碼生成**：基於 Qwen2.5-Coder 的智能程式碼補全與生成
- **沙盒驗證**：Docker 隔離環境自動執行與測試生成的程式碼
- **微調訓練**：整合 LLaMA-Factory 支援 SFT 微調流程
- **Web UI**：直覺的操作介面，降低使用門檻

## 技術棧

| 元件 | 技術 |
|------|------|
| 後端 | Python + FastAPI |
| 模型 | Qwen2.5-Coder |
| 容器化 | Docker |
| 微調 | LLaMA-Factory |

## 工作流程

```
使用者輸入 → Qwen2.5-Coder 生成 → Docker 沙盒執行 → 回傳結果
```

## 亮點

- 沙盒隔離確保生成程式碼的安全性
- 支援從生成到驗證的完整閉環
- 可擴展至更多程式語言與框架
